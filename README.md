# README

A small front-end remake inspired by the [Wolfgang Digital](https://wolfgangdigital.com?utm_source=chatgpt.com) awards page.
Built with React, Tailwind, and smooth UI interactions focused on modern agency-style design.

## Tool Used

- React
- Vite
- Tailwind CSS
- Framer Motion
- Swiper.js
- Lucide React
- React Icons
- Codex

## Features

- Responsive layout
- Animated marquee footer
- Hover image previews
- Award filtering
- Smooth transitions & micro interactions
- Contact form UI

## Run Locally

```bash
npm install
npm run dev
```

## Notes

This project was built as a front-end practice project to improve:

- component structure
- filtering
- responsive design
- animation work
- cleaner UI implementation
- React + Tailwind workflow
